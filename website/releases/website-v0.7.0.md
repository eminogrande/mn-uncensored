# ABLITERATED.cloud website-v0.7.0

This release makes exact model identity the public interface and places a much
denser interactive brain behind the cleaner hero.

## Exact model names

- The complete Hugging Face `owner/repository` values are now the primary API
  model IDs in the landing page, OpenAPI specification, agent-readable files,
  WebMCP catalog and client configuration.
- The old `mn/god`, `mn/code`, `mn/fast`, `mn/ornith-397b`, and
  `nuri/ornith-397b-abliterated` values remain compatibility aliases only.
- Prepared Modal app names are derived from the real repository and model
  names instead of the former role labels.

## Cleaner hero

- The headline is now “Intelligence, freed. Uncensored, abliterated AI.”
- The supporting introduction is “Abliterated open models in the cloud. One
  simple API.”
- The interactive brain uses 820 deterministic points and shorter connections,
  sits behind the hero copy, and still illuminates local neural regions from
  pointer movement.
- Internal endpoint status labels and the “Move to illuminate” annotation were
  removed from the public product page.

## Safe preparation only

This website release does not deploy Modal applications, download model
weights, or start GPUs. The Ornith 1.0 397B W4A16 profile remains explicitly
deployment-disabled and requires a separate signed, budget-approved runtime
release.
