FAQ

Questions, answered plainly.

What does “abliterated” mean?+

It is the upstream checkpoint label for models whose refusal behavior was reduced through abliteration. It does not guarantee zero refusals, correctness, safety, or fitness for every request.

Is the 397B model available?+

It is a real fourth catalog route with API ID mn/ornith-397b and legacy alias nuri/ornith-397b-abliterated. The currently deployed v0.3.1 gateway still exposes three routes. The next release is budget-gated and validates 397B text, vision, tool calls, shutdown, and scale-to-zero.

Do model weights download to my computer?+

No. Hugging Face stores the source artifacts, and Modal volumes cache them in the cloud. Your local machine runs only the lightweight CLI and client tools.

Does every user increase the hourly price?+

Not automatically. Users share a model container, but more requests can keep it active longer. Different routes can also run at the same time, so the workspace needs a real hard budget.

Is this a complete OpenAI API replacement?+

No. The directly tested contract is OpenAI Chat Completions with model listing, streaming, and structured tool calls. Proprietary client features may still use their original providers.

Is the project open source?+

Yes. Our code is public under Apache-2.0, an open-source license. Each model artifact keeps its own upstream license and commercial-use caveats.

Is this already a public paid platform?+

Not yet. It is a private evaluation service. Customer accounts, quotas, metering, billing, reconciliation, and commercial license review are roadmap work.
