# ABLITERATED.cloud website-v0.1.0

The first public, open-source ABLITERATED.cloud website release.

## Included

- Original responsive landing page with no copied receive.link code, assets,
  fonts, icons, or text.
- Exact four-model catalog with Hugging Face links, deployment status,
  configured context, hardware, and transparent Modal base GPU estimates.
- WhatsApp access request link for the private beta.
- Dependency-free HTML, CSS, and a small deferred navigation script.
- Open Graph image, favicon, web manifest, structured data, sitemap, crawler
  policy, and accessible responsive layouts.
- Agent-readable Markdown, `llms.txt`, `llms-full.txt`, `auth.md`, OpenAPI,
  API catalog, security contact, Content Signals, and current plus legacy Agent
  Skill discovery.
- Signed branch-based GitHub Pages deployment that cannot deploy Modal or wake
  a GPU.

## Verification

- Complete repository test suite passed.
- Secret scan passed.
- JSON, XML, internal-resource, discovery-digest, and payload checks passed.
- Desktop and responsive mobile renderings were visually inspected.
- No analytics, third-party runtime resources, webfonts, API polling, or Modal
  calls are loaded by the homepage.
