# ABLITERATED.cloud website v0.8.1

This visual refinement makes the hero brain more densely connected and the
headline easier to read.

## More synapses

Each point can now form up to ten short local connections instead of six, and
the connection radius increases from `0.145` to `0.165`. The result is a much
denser neural mesh without adding a framework, library or remote asset.

## Cleaner copy

A directional background-colored overlay fades the left side of the brain
beneath the headline, lead and actions. The fade clears gradually toward the
middle and leaves the right hemisphere crisp.

## Runtime safety

This is a static website release. It does not import or deploy Modal runtime
definitions, wake a model, or start a GPU.
